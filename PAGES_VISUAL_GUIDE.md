## Design System Reference

### Colors
```
Primary Actions: Accent (#2563EB blue)
Warnings: Warning (#F59E0B orange)
Success: Success (#10B981 green)
Errors: Danger (#EF4444 red)
Neutral: (#6B7280 gray)

Backgrounds:
- bg-white: Card backgrounds
- bg-neutral-50: Secondary backgrounds
- bg-accent-50: Light accent backgrounds
```

### Spacing System
```
Padding: px-3, px-4, px-6 (horizontal)
         py-2, py-3, py-4 (vertical)
Gap: gap-2, gap-3, gap-4 (between elements)
```

### Typography
```
Headings:
- text-5xl font-bold (page titles)
- text-4xl font-bold (section titles)
- text-lg font-semibold (card titles)

Body:
- text-base (normal text)
- text-sm (secondary text)
- text-xs (labels, badges)
```

### Hover & Active States
```
Cards: hover:shadow-md, hover:-translate-y-1
Buttons: hover:scale-[1.02], active:scale-[0.98]
Badges: pulse animation on hover (optional)
Tabs: smooth color transition + underline animation
```

### Transitions
```
All transitions use: duration-normal (150ms)
Easing: ease (default cubic-bezier)
```
